Place UI icon assets here.

Recommended usage:
- static SVG icons for simple buttons;
- Lottie JSON icons for animated buttons or state transitions;
- optional companion bitmap assets next to the JSON if an icon uses external images.

Suggested naming:
- `play.svg`
- `pause.svg`
- `play-pause.json`
- `save.json`
- `zoom-in.svg`

Preview server serves this directory at `/icons/...`.
